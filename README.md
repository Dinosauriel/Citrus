# Citrus
Having fun building a game using Rust and Vulkan!
## Getting Started (Windows):
Dependencies:
- Rust (1.77.2)
- Vulkan SDK (1.3.268.0)

Compile the source:
```
cargo build
```
Compile shaders:
```
.\compile_shaders.bat
```

Run:
```
cargo run
```
